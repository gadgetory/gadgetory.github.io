CNET How To - Use Knock Code on the LG G Pro 2

2014-03-12
I'm dan Graziano and I'm here with the latest flagship smartphone from LG the G pro two similar to the g2 the G pro 2 includes high-end specs rear facing buttons and the company's unique knock-on feature which allows the phone's display to be powered on simply by double tapping the screens the device is also the first one to include LG's new NOC code feature which uses the knock on tapping action to not only turn off the display but also to unlock the phone the first time you power on the device you'll be asked if you would like to step your NOC code if you skip this the feature can be enabled by going to the display settings clicking on lockscreen followed by the Select screen lock option and then choosing NOC code from there you'll be asked to setup a code NOC codes can range from two to eight taps in fact according to LG there are more than 80,000 different possible combinations once you confirm your code you'll be asked to create a backup pin code this will be used in case you ever forget your NOC code if you forgot both your code and your pin code you can sign in using your Google account or special tool from LG to unlock the device your NOC code can be performed anywhere on the display from a small space or across the entire width of the screen do whichever feels more natural to you and don't worry there's no need to rush out and purchase a G pro 2 if you own another LG handset the company is said that the NOC code feature will be coming to some older devices with an upcoming software update for more how-to like this one be sure to check out how to cnet.com and feel free to reach out to me on Twitter with questions or comments I'm dan Graziano CNET thanks for watching
